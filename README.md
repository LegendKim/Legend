# Legend (Unity 3D)
Unity 기반으로 개발한 프로젝트의 **소스코드 포트폴리오 저장소**입니다.  
저작권 문제로 인해 리소스(이미지, 사운드, Asset Store 애셋)는 제외하고 **스크립트와 구조만 공개**합니다.

# 프로젝트 개요
설명: 몬스터를 잡고 레벨업을 통해 어빌리티를 얻어 스테이지를 진행하는 게임
개발기간: 2020.02 ~ 2020.03  
인원: 1인 개발
엔진: Unity (C#)

# 구현 기능
- 테이블 기반 데이터 세팅
- Object Pooling으로 최적화
- AStar 알고리즘 구현
- 몬스터 AI (패트롤/추격/공격)
- 유니티 광고 및 보상 기능
